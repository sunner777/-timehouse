# 时光家项目 - 后端技术栈选择

## 1. 核心技术栈（实际使用）

| 组件 | 技术选型 | 版本 | 说明 |
|------|----------|------|------|
| **运行时** | Node.js | 20.x (ECS) | I/O密集型，处理照片上传和存储 |
| **Web框架** | Express.js | 4.18.2 | 轻量级RESTful API |
| **数据库** | MySQL | 8.0 (Docker) | 唯一数据库，users/families/family_members/photos 四张表 |
| **缓存** | Redis | 7.0 (Docker) | JWT 黑名单、SMS 限频 |
| **认证** | JWT | jsonwebtoken 9.0.2 | Bearer Token，7天过期，Redis 黑名单吊销 |
| **对象存储** | 火山引擎 TOS | SDK 2.1.0 | 照片存储，私有桶，预签名URL访问 |
| **短信** | 阿里云 SMS | dysmsapi20170525 | RequiredPhoneCode/ValidPhoneCode 认证 |

> **注意**: MongoDB 曾列入技术栈但从 V1.0 起未实际使用，已于 2026-06-27 移除依赖和相关代码。

## 2. 云服务与部署（实际架构）

| 层面 | 实际方案 | 说明 |
|------|----------|------|
| **服务器** | 阿里云 ECS 99计划 2C2G 3M | 单机部署，Ubuntu 22.04 |
| **数据库** | Docker MySQL 8.0 + Redis 7.0 | ECS 本机运行 |
| **存储** | 火山引擎 TOS（华东2上海） | 私有桶 timehouse-photos-cn-shanghai |
| **进程管理** | PM2 fork 单实例 | NODE_ENV=production |
| **反向代理** | Nginx + Let's Encrypt | 80→443 HTTPS，HTTP/2 |
| **域名** | api.timehouse.top | ICP 备案：浙ICP备2026043339号 |
| **部署** | Git push + deploy.sh | 本地 git push server → 服务器 git pull + PM2 reload |

## 3. 项目结构（实际目录）

```
backend/
├── src/
│   ├── config/           # 配置（数据库、Redis、JWT、CORS、SMS限频）
│   ├── controllers/       # 控制器（auth/photo/family）
│   ├── middleware/        # 中间件（auth JWT+黑名单, errorHandler, rateLimiter, validate Joi）
│   ├── models/mysql/      # 数据模型（User, Photo, Family, FamilyMember）
│   ├── routes/            # 路由（/api/v1/auth, /photos, /families）
│   ├── services/          # 业务逻辑（auth, photo, family, sms, tos）
│   ├── utils/             # 工具（response, codeGenerator, auditLog, sanitize）
│   ├── validators/        # Joi schema 定义（auth, photos, families）
│   └── app.js            # 入口
├── nginx/timehouse.conf  # Nginx 配置（安全头+反代）
├── ecosystem.config.js   # PM2 配置
├── deploy.sh             # Git-pull 部署脚本
├── scripts/              # 数据库初始化/迁移脚本
└── package.json
```

## 4. 安全措施（已实施）

| 层面 | 措施 |
|------|------|
| **传输** | TLS 1.2/1.3, HSTS, Let's Encrypt 自动续期 |
| **认证** | JWT Bearer Token, Redis 黑名单吊销, bcrypt cost 12 |
| **安全头** | helmet (Express) + Nginx: X-Frame-Options, X-Content-Type-Options, Referrer-Policy, Permissions-Policy |
| **输入校验** | Joi schema 路由级校验, 自动 strip unknown fields |
| **输入清理** | sanitize.js: 去控制字符/HTML标签/bidi覆盖字符 |
| **限流** | express-rate-limit (100req/15min IP) + Nginx limit_req (30r/s) + Redis SMS 限频 (60s/手机号) |
| **日志** | 生产环境不记录请求体和 Authorization 头 |
| **审计** | auditLog.js: 照片查看/删除、成员变更、权限修改 |
| **证书** | Flutter 端证书固定 (api.timehouse.top) |
