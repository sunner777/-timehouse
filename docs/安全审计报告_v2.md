# 时光家安全隐私复查 — 2026-06-13

## 🔴 需要立即修

### 1. Node.js 绑定到 0.0.0.0

**文件**：`backend/src/app.js`

```js
app.listen(config.app.port, () => {...});  // 默认绑定 *
```

当前安全组挡住了 3000 端口，但如果有天规则被意外修改，API 直接暴露。应显式绑定 `127.0.0.1`。

**修复**：`app.listen(config.app.port, '127.0.0.1', ...)`

## 🟠 需要注意（中风险）

### 2. JWT Token 明文存储（Flutter SharedPreferences）

Android 上 `SharedPreferences` 是 XML 文件，root 后可读。iOS 上是 plist。Token 泄露 = 7 天访问权限。

**缓解**：Token 黑名单已实现（上次审计修复）。登出时吊销。

### 3. 本地照片缓存无加密

`PhotoCache` 把照片存到 `getApplicationDocumentsDirectory()/photos/`。文件系统层面可被 root 用户 / 文件管理器读取。

**缓解**：照片本身来自家庭共享，设备持有者本就能看到这些照片。风险低。

### 4. 手机号明文存储

`StorageService.saveUserInfo` 把 phone 存到 SharedPreferences。设备被 root 后可读取。

**缓解**：Profile 页面已对 UI 做了脱敏显示（153 **** 4275）。
