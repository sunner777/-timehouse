# 时光家 V2.0 UI 优化方案 — 极简跨平台风格

## 设计原则

| 原则 | 说明 |
|------|------|
| **跨平台统一** | 一套代码，iOS/Android 视觉一致，不分裂 |
| **内容优先** | 照片是绝对主角，UI 退后 |
| **呼吸感** | 大留白、大圆角、信息少而精 |
| **柔性色彩** | Material 3 动态取色 + 统一色板 |
| **微交互** | 弹性动画、触觉反馈、状态过渡 |

## 技术选型

| 组件 | 选择 | 原因 |
|------|------|------|
| 设计系统 | **Material 3** | Flutter 原生支持，双平台一致 |
| 图标 | **Material Icons Rounded** | 比默认 Filled 更柔和，双平台好看 |
| 导航 | 保持现有 GoRouter + BottomNavBar | 已实现，只需美化 |

## 全局改动

### 色彩（Material 3 ColorScheme）

```dart
ColorScheme(
  brightness: Brightness.light,
  primary: Color(0xFF3B6F5A),       // 森林绿 — 家庭、自然、温暖
  onPrimary: Colors.white,
  surface: Color(0xFFFAFAF8),       // 微暖白背景
  surfaceContainerLow: Color(0xFFF3F3F1), // 卡片底色
  outline: Color(0xFFE8E8E4),       // 极淡边框
  error: Color(0xFFC62828),         // 删除等危险操作
)
```

替代之前的 iOS 蓝，用**森林绿**更契合"家庭相册"的情感定位。

### 字体（双平台通用）

| 用途 | 字号 | 字重 |
|------|------|------|
| 页面大标题 | 28pt | `w700` |
| 区块标题 | 20pt | `w600` |
| 正文 | 15pt | `w400` |
| 辅助文字 | 13pt | `w400` |
| 按钮文字 | 16pt | `w600` |

### 全局视觉规则

- **卡片圆角**：16pt（`BorderRadius.circular(16)`）
- **按钮圆角**：26pt（胶囊形 `StadiumBorder`）
- **照片网格圆角**：10pt
- **页面水平边距**：20pt
- **组件间距**：16pt
- **分割线**：极细 0.5px，颜色 `outline`，能不用就不用
- **阴影**：`elevation: 0` 为主，卡片用 1-2

## 六、交互细节

| 交互 | 实现 | 双平台表现 |
|------|------|-----------|
| **下拉刷新** | `RefreshIndicator` | iOS 转圈 / Android 圆圈 |
| **触觉反馈** | `HapticFeedback.lightImpact()` | iOS Taptic / Android 震动 |
| **弹性动画** | `spring` 曲线 | 一致 |
| **Hero 过渡** | Flutter `Hero` widget | 一致 |
| **骨架屏** | `shimmer` 包 | 一致 |
| **底部弹出** | `showModalBottomSheet` + 圆角 | 一致 |
