# 时光家 V1.0 功能需求列表

## 应用定位

家庭相册共享 App。家人拍照后上传到家庭共享组，成员随时查看彼此的照片。

---

## 一、登录

### A1 验证码登录

- **入口**：App 启动 → 未登录 → 登录页
- **流程**：
  1. 输入手机号
  2. 点击「发送验证码」
  3. 按钮变为 60s 倒计时
  4. 输入收到的验证码（4~6 位数字）
  5. 点击「登录」
  6. 登录成功，跳转首页
- **规则**：
  - 新用户自动注册
  - 同手机号 60s 内不能重复发送
  - JWT Token 有效期 7 天，期间免登录

### A2 登录状态保持

- Token 存储在本地
- 7 天内打开 App 自动进入首页，无需重新登录

### A3 退出登录

- 入口：个人中心 → 退出登录
- 确认后清除 Token，返回登录页

---

## 二、首页：共享组（Tab 1）

### B1 家庭组列表

- 顶部横向标签栏，显示用户加入的所有家庭组
- 左右滑动或点击切换当前家庭组
- 空状态：未加入任何家庭组时，提示「创建或加入一个家庭组」

### B2 照片网格

- 当前选中家庭组的照片，按日期分组展示
- 3 列网格布局
- 每张照片显示缩略图
- 空状态：家庭组内无照片时，显示「暂无照片」

### B3 上传照片

- 右下角「+」浮动按钮
- 点击 → 调起系统相册 → 选择照片 → 上传到当前家庭组
- 上传过程显示进度

### B4 照片详情

- 点击照片 → 全屏大图查看
- 左右滑动切换上一张/下一张
- 底部显示拍摄日期、上传者
- 右上角删除按钮（仅上传者和管理员可见）

---

## 三、我的空间（Tab 2）

### C1 子标签切换

- 顶部两个子标签：「照片」|「共享组」

### C2 我的照片

- 默认显示「照片」子标签
- 列出自己上传的所有照片
- 按日期分组，3 列网格
- 支持长按进入多选模式

### C3 多选操作

- 长按照片 → 进入选择模式
- 可多选照片
- 底部操作栏：「批量删除」|「修改归属」
- 选择家庭组后可批量转移照片归属

### C4 我的共享组

- 切换到「共享组」子标签
- 列表形式展示加入的所有家庭组
- 每行显示：组名、成员数、自己角色

---

## 四、个人中心（Tab 3）

### D1 个人信息

- 显示头像（默认占位图）
- 显示昵称
- 显示手机号

### D2 退出登录

- 按钮，红色文字
- 点击弹出确认对话框
- 确认后清除登录状态，跳转登录页

---

```
┌──────────┬──────────┬──────────┐
│  共享组   │ 我的空间  │   我的   │
│  🏠      │  📁      │  👤     │
└──────────┴──────────┴──────────┘
```

- Tab 1「共享组」— 家庭照片流 + 上传
- Tab 2「我的空间」— 个人照片 + 共享组管理
- Tab 3「我的」— 个人信息 + 退出

---

## 六、家庭组管理（从入口进入）

### F1 创建家庭组

- 入口：家庭组列表旁「+」或空状态引导
- 输入组名 → 创建成功 → 自动成为群主

### F2 加入家庭组

- 入口：家庭组列表旁「+」或空状态引导
- 输入 6 位邀请码 → 加入成功

### F3 成员列表

- 点击家庭组标题 → 进入成员列表
- 列表显示：头像、昵称、角色标签

### F4 邀请成员

- 成员列表页 → 邀请按钮
- 显示当前家庭组的 6 位邀请码
- 支持复制/分享

### F5 添加成员

- 群主/管理员操作
- 输入手机号 → 直接添加

### F6 权限管理

- 点击成员 → 修改角色
- 角色：群主 / 管理员 / 成员 / 游客

| 角色 | 查看照片 | 上传照片 | 删除他人照片 | 管理成员 | 解散组 |
|------|:--:|:--:|:--:|:--:|:--:|
| 群主 | ✅ | ✅ | ✅ | ✅ | ✅ |
| 管理员 | ✅ | ✅ | ✅ | ✅ | ❌ |
| 成员 | ✅ | ✅ | ❌ | ❌ | ❌ |
| 游客 | ✅ | ❌ | ❌ | ❌ | ❌ |

### F7 移除成员

- 群主/管理员可移除成员
- 确认后该成员退出家庭组

## 七、状态说明

每个页面需考虑的四种状态：

| 状态 | 表现 |
|------|------|
| **加载中** | 骨架屏或 loading 动画 |
| **正常** | 正常展示数据 |
| **空数据** | 空状态插图 + 引导文案 |
| **错误** | 错误提示 + 重试按钮 |

---

## 八、不需要设计的功能

以下是 V1.0 **不包含**的功能，墨刀原型中无需体现：

- 密码登录 / 密码注册（已废弃）
- 消息推送 / 通知
- 评论 / 点赞
- 照片编辑 / 滤镜
- 视频上传
- 搜索
