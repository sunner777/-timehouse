# 下阶段开发计划

## 开发顺序

| 阶段 | 内容 | 依赖 | 优先级 |
|------|------|------|------|
| 1 | 后端上云（ECS + RDS + HTTPS） | 无 | 🔴 最高 |
| 2 | 验证码登录（SMS + 验证码登录/注册） | 阶段 1 | 🟡 高 |
| 3 | 消息通知（短信邀请 + App内通知） | 阶段 2 | 🟡 高 |
| 4 | 前端 UI 优化（视觉统一、动效） | 无 | 🟢 中（可并行） |
| 5 | 数据隐私与安全加固（加密、审计、合规） | 阶段 1-3 | 🟢 中 |
| 6 | 性能优化（懒加载、缓存、缩略图） | 阶段 1-5 | 🔵 可穿插 |
