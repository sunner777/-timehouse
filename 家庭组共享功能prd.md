# 家庭组共享功能详述

## 0 整体页面结构

App 采用底部 3 Tab 导航，职责分明：

| Tab | 图标 | 名称 | 定位 | 核心功能 |
|---|---|---|---|---|
| 1 | group | 共享组 | 浏览共享内容 | 顶部横向 Tab 切换家庭组 → 照片时间线 → FAB 上传 → 大图浏览 |
| 2 | photo_library | 我的空间 | 管理我的内容 | 双子 Tab「照片」\|「共享组」→ 长按多选 → 批量删除/修改可见范围 |
| 3 | person | 我的 | 账户与设置 | 个人信息查看编辑、修改密码、退出登录；后续扩展设备管理、缓存清理、意见反馈、隐私政策 |

**设计原则：**
- Tab 1「共享组」= 浏览视角，快速切换家庭组看照片，家庭成员共用
- Tab 2「我的空间」= 管理视角，管理我上传的所有照片和我加入的共享组
- Tab 3「我的」= 账户视角，个人资料和 App 基础设置（V1.0 已实现）

## 1 用户路径

### 路径A：首次使用（短信登录）

1. 下载 App，进入登录页
2. 输入手机号 → 获取验证码 → 输入验证码（新用户自动注册）
3. 进入 Tab 1「共享组」，空状态引导创建或加入共享组

### 路径B：创建共享组并添加成员

1. 用户在 Tab 2「我的空间」→「共享组」子 Tab
2. 点击「创建新共享组」
3. 输入组名（2-20字）
4. 创建成功，进入该组管理页
5. 点击「添加成员」，输入手机号，选择权限等级
6. 已注册用户直接入组；未注册用户提示「该用户未注册」

### 路径C：上传照片

1. 用户在 Tab 1「共享组」或 Tab 2「我的空间-照片」，点击右下角悬浮上传按钮（FAB）
2. 唤起系统相册，多选照片（单次上限50张）
3. 点击「完成」，进入可见范围选择
4. 勾选希望对该照片可见的共享组（可多选，不选则仅自己可见）
5. 确认上传，照片出现在「我的空间」及被选中的共享组

### 路径D：浏览照片

1. 用户在 Tab 1「共享组」，顶部横向 Tab 切换目标组
2. 上下滑动浏览时间线，照片按日期分组倒序排列，分组标题粘性吸顶
3. 照片网格：3列正方形缩略图，等间距
4. 点击照片进入大图浏览页，左右滑动切换

### 路径E：批量管理照片

1. 用户在 Tab 2「我的空间」→「照片」子 Tab
2. 长按照片进入多选模式，勾选多张照片
3. 底部出现「删除」和「共享组可见」按钮
4. 点击「删除」→ 二次确认 → 云端彻底删除
5. 点击「共享组可见」→ 弹出共享组选择器 → 勾选目标组 → 确认后所选照片对勾选的组可见

### 路径F：个人账户管理

1. 用户在 Tab 3「我的」
2. 查看/编辑头像、昵称、手机号
3. 修改密码：输入旧密码 → 输入新密码 → 确认
4. 退出登录：点击退出 → 二次确认 → 清除本地 token → 跳转登录页
5. （后续扩展）设备管理、缓存清理、意见反馈、隐私政策等入口

## 2 关键页面

### 登录页

- 手机号输入框（数字键盘，11位格式校验）
- 验证码输入框（6位数字，输满自动触发登录）
- 「获取验证码」按钮（60秒倒计时，防重复点击）

### 共享组页（Tab 1）

- 顶部横向 Tab 栏：展示所有共享组名称，左右滑动切换，默认照片最多的组排第1
- 空状态：插画 +「还没有照片，点击上传吧」
- 内容区：照片网格，按日期分组（如「2月10日」），倒序排列，分组标题粘性吸顶
- 悬浮上传按钮：右下角圆形 FAB，点击进入上传流程（路径C）
- 点击照片：进入大图浏览页

### 照片浏览页（大图）

- 全屏沉浸式，黑色背景
- 左右滑动切换照片，捏合缩放，双击放大
- 底部信息栏：上传者昵称 + 上传时间
- 底部操作栏：下载图标（有权限时显示）、删除图标（有权限时显示）

### 我的空间页（Tab 2）

- 顶部子 Tab 导航：「照片」|「共享组」
- **「照片」子 Tab**：
  - 跨组聚合展示我上传的所有照片，按日期时间线排列
  - 悬浮上传按钮（FAB）
  - 长按进入多选模式，底部出现批量操作按钮
- **「共享组」子 Tab**：
  - 我创建或加入的共享组列表
  - 顶部有「创建新共享」按钮
  - 点击组名进入该组成员管理页
  - 每个组显示：组名、我的角色、成员数

### 成员管理页

- 从「我的空间→共享组→点击组名」或「共享组页→长按/管理入口」进入
- 成员列表：头像 + 昵称 + 权限标签（创建者/管理员/成员/访客）
- 组长显示「创建者」标签
- 「添加成员」按钮（FAB，输入手机号+选择角色）
- 点击成员项：底部弹出权限修改面板（L1 可管理 / L2 可编辑 / L3 仅查看）

### 我的页（Tab 3）

- 顶部：头像 + 昵称 + 手机号（点击可编辑）
- 功能列表：
  - 修改密码
  - 退出登录
- （V4.0 扩展）设备管理、账号安全、缓存管理、意见反馈、隐私政策

## 3 字段规则

| 字段      | 类型     | 规则        | 错误提示        |
| ------- | ------ | --------- | ----------- |
| 手机号     | string | 11位数字，1开头 | 请输入正确的手机号   |
| 验证码     | string | 6位数字      | 验证码格式错误     |
| 共享组名    | string | 2-20个字符   | 组名需在2-20字之间 |
| 用户ID    | string | 系统生成的唯一标识 | 用户不存在       |
| 单次上传照片数 | int    | 上限50张     | 单次最多选择50张照片 |

## 4 异常处理

| 场景           | 处理方式                                                |
| ------------ | --------------------------------------------------- |
| 网络异常（上传中）    | 暂停上传，Toast提示「网络异常，恢复后自动续传」                          |
| 网络异常（浏览时）    | 显示已缓存缩略图，原图提示「点击重试」                                 |
| 验证码发送失败      | Toast「发送失败，请重试，60秒后可重新获取」                           |
| 添加成员：手机号已注册  | 直接加入组                                          |
| 添加成员：手机号未注册  | 提示「该用户未注册」                           |
| 添加成员：手机号格式错误 | 前端实时校验，输入框变红提示                                      |
| 删除权限不足       | 大图页直接隐藏删除按钮，不显示也不提示                                 |
| 照片上传重复       | 基于文件hash检测，提示「该照片已存在」                               |
| 存储空间不足（后期）   | Toast「空间不足，可清理照片或开通会员扩容」                            |
| 返回上一页        | 所有二级及以上页面均提供返回上一页的能力，顶部左侧显示返回箭头，遵循iOS/Android平台默认规范 |
